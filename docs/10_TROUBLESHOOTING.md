# 10 - Troubleshooting & Maintenance Guide

This document lists common errors, error codes, database issues, licensing errors, and hardware scanner/printer problems, along with instructions to resolve them.

---

## 🗄️ Database & Schema Troubleshooting

### Error: `SQLITE_BUSY: database is locked`
* **Why it occurs**: SQLite allows multiple read operations but restricts write operations to a single process. This error occurs if a database transaction is locked, or if another process (such as a database viewer or utility) is accessing the database file.
* **Resolution**:
  1. Close any database viewers (e.g. DB Browser for SQLite, DBeaver) or background diagnostic processes.
  2. If the error occurs during a backup restore, check that the restore service has closed the database connection pool using `closeDatabase()`.
  3. Restart the ERP application.

### Error: `SQLITE_CORRUPT: database disk image is malformed`
* **Why it occurs**: Severe filesystem errors, sudden power outages during write transactions, or disk write failures have corrupted the SQLite file database header.
* **Resolution**:
  1. Go to the backup directory: `C:\Users\<Username>\Documents\JewelleryERP_Backups`.
  2. Identify the latest backup file (e.g. `jewellery_erp_backup_2026-06-22.db.gz`).
  3. Close the application. Decompress the backup file using a utility like 7-Zip, rename the file to `jewellery_erp.db`, and use it to replace the corrupted database file in AppData: `C:\Users\<Username>\AppData\Roaming\JewelleryERP\db\jewellery_erp.db`.

## 🔑 Licensing & Activation Troubleshooting

### Error: `Verification failed: License key is invalid for this machine`
* **Why it occurs**: The decrypted license key payload `deviceId` does not match the active Device ID fingerprint generated by the machine's hardware. This occurs if you copy an active license key to a different computer.
* **Resolution**:
  1. In the **System Secure Lock** screen, click **Copy Device ID**.
  2. Send the copied Device ID fingerprint to the ERP system administrator to generate a valid key.
  3. Enter the new machine-bound key to activate the license.

### Error: `License key expired`
* **Why it occurs**: The subscription or evaluation license period has ended.
* **Resolution**:
  1. Generate a new license key with an updated expiration date.
  2. Enter the new license key on the secure lock screen to reactivate the software.

## 🔌 Hardware Scanner (Barcode/QR) Troubleshooting

### Issue: Scanned values do not appear in the input fields
* **Why it occurs**: The scanner connection interface is configured incorrectly, or the target input field does not have focus.
* **Resolution**:
  1. Open the **Scanner Settings** screen.
  2. Verify that **Enable Hardware Scanning System** is checked.
  3. Check the **Connection Interface**:
     - For standard USB scanners, select **USB HID Keyboard Emulation**.
     - For serial scanners, select **Virtual COM Port** and check the COM Port assignment and Baud Rate.
  4. Ensure you click inside the target input field (e.g. the **Tag No** field) before scanning.

### Issue: Scans are cut off or missing characters
* **Why it occurs**: The hardware wedge input speed is faster than the Chromium renderer process can capture, causing characters to be skipped.
* **Resolution**:
  1. Open the **Scanner Settings** screen and test the scanner in the diagnostics console.
  2. If the console logs show high inter-keystroke latency or missing characters, configure the scanner hardware:
     - Connect the scanner directly to a USB port (avoid USB hubs).
     - Adjust the scanner's hardware transmission delay settings (add a 5ms to 10ms inter-character delay).

## 🖨️ Printer & Tag Template Alignment Troubleshooting

### Issue: Printed barcodes are cut off or margins are misaligned
* **Why it occurs**: The template size and margins configured in the printer driver do not match the tag layout settings in the application.
* **Resolution**:
  1. Open the Windows **Printers & Scanners** control panel.
  2. Select the label printer (e.g., Zebra TLP2844) and click **Printing Preferences**.
  3. Set the print media layout settings (e.g. select thermal labels, set label dimensions, and choose gaps/markings).
  4. In the ERP Printer Settings screen, verify that the selected **Label Sheet Size** matches the printer's label roll.
  5. Adjust margins and scaling settings in the printer's configuration template JSON.

## ☁️ Cloud Sync & Connection Troubleshooting

### Error: `No internet connection detected`
* **Why it occurs**: The system's network sweep ping to Google DNS (`8.8.8.8`) timed out, indicating no network access.
* **Resolution**:
  1. Check the network cables and router.
  2. Verify that your firewall or anti-virus software is not blocking outgoing ICMP ping requests or the Electron network connection.
  3. Try running `ping 8.8.8.8` in PowerShell to test your connection.

### Error: `Cloud server returned error code: 401 Unauthorized`
* **Why it occurs**: The cloud synchronization token (`cloudSyncToken`) configured in Company Settings is invalid or has expired.
* **Resolution**:
  1. Open the **Company Settings** screen.
  2. Enter a valid cloud sync token in the configuration fields.
  3. Click **Save Settings** and retry the synchronization.
